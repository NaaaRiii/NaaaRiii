## サービス概要
仕事や日々の研鑽を重ねていくのと同時に、それが数字として可視化できたら面白いなと思いこのサービスを開発しました。やるべきことを記録しそれを達成すると経験値が貯まっていく、RPGゲームのようなものを作ればもっと日々が楽しくなると感じています。
